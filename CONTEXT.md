# CONTEXT - 안상원

## 기본 정보
- 나이: 1986년생 (39세)
- 직업: 회사원
- 가족: 기혼, 둘째딸 출산 예정

## 현재 목표
- **핵심 목표**: 경제적 자유
- **전략**: 매일 5% 복리 복리의 마법
- **시드머니**: 100만원 (코인 선물 계정 $200 포함)
- **투자 수단**: 코인 선물 + 주식

## 투자 현황
- 코인 선물 계정: $200 (5% 복리 전략 주 운용)
- 주식: 예정

## 알려진 약점 (반드시 체크)
- **뇌동매매**: 비이성적인 충동 매매 반복
- **손절 실패**: 수익을 지키지 못하고 청산당하는 패턴 반복
- 감정이 개입되면 원칙을 무너뜨리는 경향 있음

## 관심 분야
- 코인 선물 트레이딩
- 주식 투자
- 경제적 자유 / 파이어(FIRE)
- 복리 투자 전략

## Claude에게 당부
- 투자 관련 얘기 나오면 반드시 약점 패턴 체크
- 뇌동매매 징후 보이면 즉시 지적
- 동의보다 반박 먼저
- 감정적 판단과 논리적 판단을 구분해서 피드백
